# Spark Iceberg Docker Quickstart

## Build the Spark Iceberg Docker image

```sh
docker-compose build --no-cache
```

## Run the Spark Iceberg Docker image

```sh
docker-compose up -d
```

## Access the URLs

### Notebooks

```sh
http://localhost:8888/tree
```

### Spark Master

```
http://localhost:8080/
```

### Spark History Server

```sh
http://localhost:18080/
```

### Mini IO

```sh
http://localhost:9001/
```
